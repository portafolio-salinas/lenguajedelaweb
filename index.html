    HTML5( Hyper Text Markup Languaje, vercion5) es la quinta versión importante del lenguaje básico de la World Wide Web, HTML.

    Hoja de estilo en cascada o CSS (siglas en ingles de cascadyng style sheets) es un lenguaje usado para definir y crear la presentación de un documento estructurado esrito en HTML.

    Usa la extencion .css

    JavaScript(Abreviada comunmente "JS") es un lenguaje de programacion interpretado utilizando principalmente por el cliente (client-side) implementando como parte de un navegador web permitiendo mejoras en la inerfaz de usuario y páguina web dinámica.

    Usa la extención .js
